After comparing a couple of free web hosting sites suggested in 7 “Best” Free Web Hosting Sites (2019) I chose InfinityFree.net and used it to host the WordPress site.

* 2. Move WordPress from localhost to live server

1) Tried the Duplicator plugin by following the instruction from How to Move WordPress From Local Server to Live Site (2 Methods)

But when connected to the live site using an FTP client, there was an error and hence no file could be upload from local server to live site

Even though I tried to learn a bit of FTP from the online torurial How to connect to your WordPress Website with FTP

2) Tried another plugin All-In-One WP Migration by following the instruction from How to Move WordPress from Localhost to Live Server : Beginners Guide

However, the imported file exceeds the maximum upload size for this site: 10 MB

So here's the solution to How to Increase Maximum Upload File Size in WordPress

  • Download the plugin All-in-One WP Migration Import
  • Installed the above plugin by following the instruction from Install Instructions for File Extension
  • Finally, the site was moved from local host to a live site
